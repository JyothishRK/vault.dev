TCP v/s UDP:
-----
TCP:
---
* Reliable, ordered.
* Slower (For confirmation part)
* If a packet drops, it's resent
* Example: SSH, Email, Web.

UDP:
---
* Fire and forget
* Faster
* If a packet drops, it's lost.
* Example: Video streaming, Online gaming, DNS queries. ^xM6qONjO

Data communication:
-----
* Exchange of data between 2 devices. ^xG1xjLSK
